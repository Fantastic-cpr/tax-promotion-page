# 税务宣传项目部署指南

## 项目简介

这是一个系列情景剧宣传网页项目，采用 React + TypeScript 构建，模拟微信公众号文章风格，提供税务宣传内容的展示功能。

## 目录结构

```
├── src/              # 源代码目录
│   ├── pages/        # 页面组件
│   ├── components/   # 通用组件
│   ├── contexts/     # React 上下文
│   ├── hooks/        # 自定义钩子
│   └── lib/          # 工具函数
├── index.html        # HTML 入口文件
└── package.json      # 项目配置和依赖
```

## 快速开始

### 前提条件

- 安装 Node.js 16.x 或更高版本
- 安装 npm 8.x 或更高版本

### 安装依赖

```bash
pnpm install
```

### 开发模式运行

```bash
pnpm dev
```

项目将在 http://localhost:3000 启动开发服务器

### 构建生产版本

```bash
npm run build
```

构建后的文件将位于 `dist/static` 目录中

## 微信公众号集成

本项目已针对微信公众号环境进行了优化：

1. 添加了微信分享所需的元数据
2. 实现了微信风格的界面设计
3. 支持在微信环境中使用分享功能
4. 针对微信浏览器进行了样式适配

## 项目功能

- 税务宣传内容展示
- 微信公众号风格界面
- 阅读进度追踪
- 文章目录导航
- 响应式设计，适配不同设备
- 分享、点赞等社交功能

## 技术栈

- React 18+
- TypeScript
- Tailwind CSS
- Vite 构建工具
- React Router
- Framer Motion 动画

## 注意事项

1. 项目中的图片使用了生成服务，请确保网络连接正常
2. 在微信环境中使用时，可能需要进行微信公众号的相关配置
3. 如需修改项目配置，请参考相关配置文件
